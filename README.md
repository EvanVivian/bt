#### 项目简介

该项目是基于python的web类库django开发的一套web网站，做为本人的毕业设计。
本人的研究方向是一项关于搜索的研究项目。在该项目中，笔者开发了一个简单版的搜索网站，实现了对数据库数据的检索和更新。通过开发该项目，笔者学习和巩固了python的相关技术和框架。
